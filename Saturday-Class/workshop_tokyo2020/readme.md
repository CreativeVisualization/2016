# 東京オリンピック・パラリンピック2020 エンブレムをコードする

## アウトライン
### 目的

正式決定した東京オリンピック・パラリンピックのエンブレムを題材として、アルゴリズムによる造形について研究する。

### どんなことをするか

* エンブレムをコードで記述し再現する
    * ルールや規則性を見つける
    * パラメトリックな部分を見つける
* デザインツール化する。
	* ルールやパラメータに変更を加えて、色や形が異なるバリエーションを作る
	* 誰にでも使いやすいようにデザインツール化する
